사용 설명서:


Main Menu

 -Network Settings
   
    1)Set Static IP: 고정 IP 설정하기
    2)Reset: 유동 IP(DHCP4) 설정하기


 -Samba Settings

    1)Add samba: 삼바 설정 추가하기
    2)Edit Samba: 삼바 설정 수정하기
    3)Delete Samba: 삼바 설정 삭제하기
    4)Samba User Conf

      a)Add User: Samba 사용자 추가
      b)Del user: Samba 사용자 삭제


 -Exit: 프로그램 종료




이동방법- 상하 방향키: 위아래 움직이기
          좌우 방향키: 뒤로가기/선택 (텍스트 수정모드 일때는 커서 좌/우로 움직임)
          엔터: 선택

기타 키 조합- CTRL+E :프로그램 강제 종료
              F(5): 뒤로 가기
              F(7): 프로그램을 통해 설정한 모든 삼바, 네트워크 설정 초기화
